### To view the project

° Download or clone the repository on your machine.

° Open the terminal in the project directory and run:

`$ npm install`

or

`$ yarn add`

Then:

`$ npm run dev`

or

`$ yarn run dev`

° Click on the terminal link or open your browser in http://localhost:5173/
